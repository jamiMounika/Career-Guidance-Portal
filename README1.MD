# Career Guidance and Recommendation Portal

## Project Overview

This project helps users discover suitable career paths based on their:

- Hard Skills
- Soft Skills
- Interests
- Hobbies
- Strengths
- Weaknesses
- Experience (Optional)

The system analyzes user inputs and recommends suitable careers from 100+ career profiles.

## Features

- Career Assessment Form
- 100+ Career Profiles
- Multiple Career Categories
- Match Percentage
- Career Recommendations
- Education Path Suggestions

## Technologies Used

- HTML
- GitHub Pages
